What is this?

These are some of the things I've seen.

I love gifs and the way they can play back to others. This collection is hopefully a way of making use of all the digital material I've accrued in a way that others can benefit and be entertained.

If you wanted to make us of this please do so, and if you wanted to say thanks feel free to buy me a coffe. If you're interested I have more static moments available here.

How was this made?

  • The photos are taken sequentially usually in one moment on my phone, currently an iphone.
  • The photos are stitched together either on my phone using a shortcut or 5secondsApp or alternatively on a desktop using PhotoScape.
  • This is hosted on github pages, a fast guide to setting a page up has been created by Daniel Shiffman here.
  • Images are hosted and served using Image kit.
  • This makes use of the awsm css library.
